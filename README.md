# 🌍 terramind-core - Real-time disaster intelligence for everyone

[![](https://img.shields.io/badge/Download-Latest_Version-blue.svg)](https://github.com/chamind5843/terramind-core/releases)

Terramind-core tracks natural disasters across the globe. It gathers data from agencies like NASA, USGS, and NOAA to provide a clear view of earthquakes, wildfires, and weather threats. You can monitor active events, understand regional risks, and receive updates through an integrated dashboard.

## 📥 How to Install

Follow these steps to set up the software on your Windows computer.

1. Visit the [Download Page](https://github.com/chamind5843/terramind-core/releases).
2. Look for the "Assets" section at the bottom of the latest release.
3. Select the file ending in `.exe` to begin the download.
4. Open the downloaded file once the process finishes.
5. Follow the prompts on your screen to complete the installation.
6. Launch the application from your desktop or Start menu.

## 🚀 Key Features

This software combines multiple data sources into one interface. You avoid the need to visit separate government websites to track hazards.

- **Unified Intelligence**: The application fuses information from USGS, NASA, and NOAA into a single map.
- **Active Monitoring**: See earthquake magnitude, wildfire perimeters, and severe weather watches in real time.
- **GeoScience Assistant**: Use the AI-powered assistant to ask questions about specific geographic regions or past disaster events.
- **Exposure Analysis**: View data from the GlobalBuildingAtlas to understand how physical infrastructure relates to regional hazards.

## 🖥️ System Requirements

Ensure your computer meets these standards for a stable experience.

- **Operating System**: Windows 10 or Windows 11.
- **Memory**: At least 8GB of RAM.
- **Storage**: 500MB of free disk space.
- **Connection**: An active internet connection for real-time data updates.
- **Display**: A screen resolution of 1920x1080 or higher ensures the interface remains readable.

## 🛠️ Using the Dashboard

The dashboard provides a visual map of the globe. Clusters of icons represent ongoing events.

- **Navigation**: Click and drag your mouse to move across the map. Use the scroll wheel to zoom in on specific zones. 
- **Filters**: Access the sidebar menu to toggle different event types. You can hide wildfire data if you only wish to observe earthquake activity.
- **Details Panel**: Click any icon on the map to open a detailed report. This panel provides the date, time, source agency, and severity of the event.
- **Search**: Enter a city or region name in the search bar to zoom directly to that location.

## 🤖 The GeoScience Assistant

The assistant helps you interpret complex data. You can type natural language queries into the chat box located at the bottom of the screen.

- **Ask for context**: Type "What is the history of wildfires in this region?" to get a summarized report.
- **Check definitions**: If you see a technical term related to seismic or weather patterns, ask the assistant to define it in plain language.
- **Request comparisons**: You can compare current weather alerts against historical records stored in the database.

## ⚙️ Settings and Customization

You can change how the software behaves to suit your preferences.

- **Theme**: Toggle between light and dark modes to reduce eye strain.
- **Units**: Set your preference for distance and temperature units, such as miles and Fahrenheit or kilometers and Celsius.
- **Alerts**: Enable desktop notifications to receive immediate information when a high-priority earthquake or weather event occurs.
- **Data Refresh Rate**: Adjust how often the application fetches updates from global servers. A shorter interval provides faster data but uses more network bandwidth.

## 🔍 Frequently Asked Questions

**Does the software require a paid subscription?**
No. Terramind-core is free to use.

**Where does the data originate?**
The data comes from official government databases, including the USGS, NASA, and NOAA.

**Does the application save my location?**
The application uses your location only to center the map on your region. You can disable this in the settings menu at any time.

**Can I export the data for my own use?**
Yes. You can export current map views or reports as PDF files through the File menu.

**What should I do if the map fails to load?**
Check your internet connection first. If the connection is stable, restart the application. If the problem persists, visit the releases page to ensure you have the most recent version of the software installed.

**Is my hardware safe while running this software?**
The application performs standard data processing tasks. It does not place excessive strain on your processor or graphics card during normal use.
